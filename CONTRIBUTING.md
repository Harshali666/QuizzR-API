### Make a contribution!

Submit a Pull Request, with your question added to the questions/index.json file. Make sure the question is in this format:

```javascript
{
  "id": last question id + 1,
  "difficulty": #(1-5),
  "question": "____?",
  "a": "",
  "b:: "",
  "c": "",
  "d": "",
  "answer": "(#)"(replace # -> option having correct answer)
}
```
